# Programming Foundations Assignment 2

## How to run
To run the calculator program, make sure all python files are in the same directory, and the pictures directory is also available.  
Once that is done, run the calc.py file.  

## Contents of repo

### calc.py
Running this file will start program.  
It opens up a menu to select from:
- Calculator
- Graphing tool
- Table and Statistis

### calculate.py
This file is for the 'Calculator' screen.  
The code in here manages the GUI and logic for calculations.  
The user is able to do:  
- Simple Arithmetic
- Trigonometry
- Exponentiation and rooting
- Logarithms
- Factorials
- Reciprocation / +- current value on the display
- Use constants e and pi
